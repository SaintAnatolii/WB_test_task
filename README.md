#  WB Users - Менеджер пользователей
##  О проекте

WB Users позволяет:

-  **Просматривать список пользователей** в удобной таблице
-  **Добавлять новых пользователей** через модальное окно
-  **Редактировать существующих пользователей** с одного клика
-  **Удалять пользователей** из системы
-  **Просматривать профили** с детальной информацией
-  **Видеть ошибки в UI** — не в консоли, а прямо на интерфейсе

##  Технологический стек

- **React 19** + **TypeScript 5.9** — фреймворк с типизацией
- **Vite** — молниеносный сборщик приложений
- **Redux Toolkit** — управление глобальным состоянием
- **Axios** — HTTP клиент для работы с API
- **React Router** — навигация между страницами
- **MockAPI** — облачное хранилище для тестовых данных
